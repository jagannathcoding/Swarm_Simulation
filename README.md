# 3D Particle Swarm Simulator

An interactive 3D particle simulation built with React and Three.js. This project combines cinematic particle systems, geometric formations, and motion-rich UI transitions to create a premium visual frontend experience.

## Tech Stack

- React
- Vite
- Three.js
- React Three Fiber
- Framer Motion
- GSAP
- react-spring

## Features

- Real-time 3D particle scenes
- Multiple visual modes including Nebula, Vortex, Helix, Prism Pyramid, Crystal Lattice, and Orbital Crown
- Smooth spring-based transitions between structures
- Motion-enhanced UI using Framer Motion and GSAP
- Fully responsive fullscreen experience
- Dark cinematic visual styling
