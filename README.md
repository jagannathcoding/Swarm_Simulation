# AI Particle Swarm Simulator

A high-end interactive 3D particle experience built with React and Three.js. This project blends cinematic particles, geometric structures, and motion-rich UI transitions to create a premium frontend visual experiment.

## Tech Stack

- React
- Vite
- Three.js
- React Three Fiber
- Framer Motion
- GSAP
- react-spring

## Features

- Real-time 3D particle scenes
- Multiple visual modes including Nebula, Vortex, Helix, Prism Pyramid, Crystal Lattice, and Orbital Crown
- Spring-driven 3D transitions between structures
- Motion-enhanced UI with Framer Motion and GSAP
- Responsive fullscreen presentation
- Dark cinematic visual styling

## Getting Started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Purpose

This project was created to explore:

- immersive 3D visuals in React
- particle-based motion systems
- cinematic UI animation workflows
- combining React Three Fiber with Framer Motion, GSAP, and react-spring
